# Roadmap: 100 блоков улучшений VoiceForge

Единый план развития по всем направлениям после реализации блоков 1–30 (Desktop GUI Phase 2). Блоки сгруппированы по доменам; порядок выполнения — по приоритету внутри группы.

**Связь:** [desktop-gui-phase2-10-blocks.md](desktop-gui-phase2-10-blocks.md) (блоки 1–20), план блоков 21–30 (single instance, deep link, store, FTS, CalDAV, updater и др.) реализован.

## Порядок реализации (рекомендуемый)

1. **Сначала быстрые победы (31–35, 51–55):** настраиваемые горячие клавиши, уведомления по анализу, CLI version/status --json, подсказки при ошибках.
2. **Затем устойчивость (61–65, 69):** D-Bus GetVersion, сигналы, таймауты, логи демона.
3. **Документация и CI (89, 92–94):** сборка десктопа в CI, runbook updater, индекс доков.
4. **Остальное** — по приоритету из плана и обратной связи.
