# Что нужно от пользователя (решения и действия вне автопилота)

Список решений, подтверждений и действий, которые **не может выполнить агент на автопилоте**: требуют явного решения владельца репозитория или пользователя.

**Обновлено:** 2026-03-14.

| Задача | Вариант A | Вариант B | Вариант C |

## 6. Краткий чеклист «что точно не сделает автопилот»

- Не разрешает KV-гейты и не начинает заблокированные KC без явной формулировки «KV разрешён».
- Не подписывает бинарники и не публикует секреты.
- Не меняет ключи в keyring и не имеет доступа к Cursor Secrets.
- Не принимает бизнес-решения (цены, упаковка, платформы).
- Не проводит живой пилот с пользователями (KV3).
- Не ставит финальный визуальный sign-off на overlay (KV2) — только документирует и готовит код.

Все остальное (код, тесты, доки, коммиты, теги, release runbook, аудит-документы) агент может делать в рамках правил репозитория и явных инструкций.
