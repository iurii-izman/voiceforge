# Интеграция с видеовстречами (Jitsi / Meet) — заглушка

**Статус:** запланировано (roadmap, блок 80). Не реализовано.

## Цель

В будущем VoiceForge может поддерживать:

- захват аудио с видеовстреч (Jitsi, Google Meet и др.);
- автоматический старт записи по календарю при входе в встречу;
- привязку сессий к событиям календаря.

## Текущее состояние

- Захват аудио идёт через **PipeWire** (системный источник).
- Календарь: CalDAV опрос, виджет «ближайшие события», контекст следующей встречи для анализа.
- Прямой интеграции с Jitsi/Meet (расширения, API) **нет**.
